# Insurance Data Analysis – Power BI Project
 ##  Project Description
This project presents a comprehensive Insurance Data Analysis solution built using Microsoft SQL Server and Power BI. The objective is to analyze insurance policies, claims, and customer-related data to gain insights into business performance, customer distribution, and trends. The project also demonstrates advanced Power BI features such as drill-through, row-level security (RLS), scheduled refresh, and sentiment analysis.

##  Tools & Technologies
•	Microsoft SQL Server
•	Power BI Desktop & Power BI Service
•	DAX
•	Power Query
•	Row-Level Security (RLS)

##  Project Implementation Steps
1️⃣ Data Setup & Integration
•	Downloaded and installed Microsoft SQL Server
•	Imported insurance data into SQL Server tables
•	Loaded data into Power BI Desktop for reporting

2️⃣ Data Understanding & Profiling
•	Reviewed data in Table View
•	Performed data profiling to check accuracy, null values, and consistency
•	Added slicers and filters for interactive analysis

3️⃣ Report Development & Visualizations
•	Created Card visuals to display KPIs
•	Added Multi-row cards for summary metrics
•	Designed a Ribbon Chart to track category-wise trends
•	Built Bar and Line Charts to analyze policy and claim performance
•	Added Donut Chart and Map Visual for geographical insights

4️⃣ Publishing & Refresh Automation
•	Published the report to Power BI Service
•	Configured scheduled refresh to keep data up to date
•	Tested refresh and republished updated reports

5️⃣ Advanced Power BI Features
•	Implemented Drill-Through Filters for detailed record-level analysis
•	Created and tested Roles in Power BI Desktop
•	Implemented and validated Row-Level Security (RLS) in Power BI Service
•	Built Power BI Dashboards from reports for executive insights

6️⃣ Sentiment Analysis
•	Performed sentiment analysis using Power Query
•	Classified customer feedback into sentiment categories
•	Added sentiment-based visuals to enhance decision-making insights
## Key Insights Delivered
•	Analysis of insurance policies and claims trends
•	Customer and region-wise performance insights
•	Secure data access using RLS
•	Real-time reporting through scheduled refresh
•	Customer sentiment insights to improve service quality

## Conclusion
This project showcases an end-to-end insurance analytics solution using Power BI, highlighting strong capabilities in data modeling, visualization, automation, security, and sentiment analysis. It demonstrates how business intelligence can support strategic decision-making in the insurance domain.
